# Operators

This folder contains programs that demonstrate various operators and conditional logic in Java.

## Files

- **ArithmeticOperators.java**: Performs multiple arithmetic operations like addition, subtraction, multiplication, division, modulus, and calculations for perimeter, area, simple interest, compound interest, and temperature conversion.
- **RelationalOperators.java**: Includes methods for checking positivity/negativity, odd/even, greatest of three numbers, leap year, grade calculation, and age categorization.
- **SwapTwoVariables.java**: Swaps the values of two variables using a temporary variable.
