# Patterns

This folder contains programs that print various star patterns using loops.

## Files

- **LeftHalfPyramid.java**: Prints a left-aligned half pyramid of stars.
- **RightHalfPyramid.java**: Prints a right-aligned half pyramid of stars.
- **ReverseRightHalfPyramid.java**: Prints a reverse right-aligned half pyramid of stars.
